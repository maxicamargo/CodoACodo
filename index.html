El camino a FullStack con Python

En este pequeño blog intentare contarles o mejor dicho mostrarles todo el camino que he tenido que recorrer para lograr convertirme o adquirir los conocimientos para ser un desarrollador FullStack.

¿Que es ser FullStack?

Un desarrollador FullStack es una persona que puede desarrollar un proyecto completo, osea conoce y entiende de FrontEnd y BackEnd, pero con la salvedad de que no es un especialista en ninguna de las dos disciplinas. Es por ello que hoy en dia es un perfil muy requerido por las empresas. Ahora bien, pero para poder tener todos estos conocimientos, y no morir en el intento, creo que lo mejor es dividir nuestras metas y objetivos en tres grandes grupos:

  • Logica de Programacion y herramientas basicas (Pseudocodigo, PSInt, Python, Git, Bases de Datos)
  • Programacion Web Inicial (HTML, CSS, Javascript, Vue, SQL, Python, Flas)
  • Programacion Web Avanzada (Django)
Cada un de estos grupos es fundamental y es necesario dominar los contenidos para lograr realizar nuestro camino sin complicaciones. Asi que sin mas preambulos comencemos a dar nuestro primeros pasos en el mundo de la programacion...

Programacion Web Inicial

Plan de estudios

  1. HTML y CSS:

    2. Aprenderás a crear tus primeras estructuras en HTML y darle estilo a tu página web con CSS.

  2. Maquetación:

    3. Aprenderás a utilizar herramientas fundamentales a la hora de estructurar tu página web.

  3. Animaciones y transformaciones:

    4. Aprenderás a crear animaciones para incorporar a tu sitio web.

  4. GIT:

    5. Podrás crear repositorios locales y remotos para alojar el código de tu proyecto web.

  5. Bootstrap:

    6. Dominarás uno de los frameworks líderes en lo que respecta a diseño de sitios web adaptables a todo dispositivo (Responsive Web Design).

  6. Javascript:

    7. Aprenderás los fundamentos de la programación con el lenguaje de programación web más utilizado en la actualidad.

  7. Vue.js:

    8. Abordarás los fundamentos para crear aplicaciones reactivas, manejar componentes, consumir servicios externos y demás.

  8. Servidores:

    9. Publicarás tus primeros sitios web en distintos servidores, así como también podrás aprender conceptos claves sobre la arquitectura web actual.

  9. Base de datos:

    10. Aprenderás a trabajar con uno de los gestores de base de datos más utilizados en la actualidad y podrás conocer la sintaxis SQL, el lenguaje fundamental para comunicarte con una base de datos.

  10. Python y POO:

    11. Lograrás programar con uno de los lenguajes más solicitados del mercado. Podrás conocer la lógica y sintaxis de este lenguaje. Descubrirás una nueva forma de pensar y crear proyectos con el Paradigma Orientado a Objetos.

  11. Framework:

    12. Aprenderás a desarrollar proyectos complejos mediante un framework Python de alto nivel el cual fomenta el desarrollo rápido de aplicaciones web, en conjunto con un diseño limpio y pragmático. Podrás escribir código que genere HTML y CSS de forma dinámica. Podrás ver las ventajas de usar un framework para desarrollar sistemas web estables, seguros y escalables desde cero.

  12. Proyecto final:

    13. Desarrollarás un proyecto final en el cual podrás integrar las tecnologías vistas a lo largo de la cursada permitiéndote construir un proyecto para incorporar a tu portafolio/CV y que te ayudará a encontrar tu primer empleo en el mundo IT.
